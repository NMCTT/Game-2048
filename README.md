├── settings.py          # Định nghĩa các hằng số: màu sắc, kích thước cửa sổ và phông chữ.<br>
├── assets/              # Thư mục chứa tài nguyên bổ trợ (font, âm thanh nếu có).<br>
└── README.md            # Tài liệu hướng dẫn và thông tin dự án.<br>
